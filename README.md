**Project_Overview**
This project focuses on manual and automated testing of the QA Brains Practice E-Commerce website (QA Brains Practice E-Commerce Website). The objective is to validate key user workflows such as authentication, product browsing, cart operations, and checkout functionality from an end-user perspective.
The testing approach combines manual testing techniques (functional, system, usability, and exploratory testing) with automation testing using Selenium WebDriver, Java, TestNG, and Apache Maven to ensure efficient and reliable test coverage.
This project includes test planning, test case design, execution, defect reporting, and automation of critical workflows, aiming to identify functional defects, UI issues, and improve overall application quality.
